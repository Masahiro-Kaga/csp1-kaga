Dive into Next.js

I often see questions about how the JavaScript libraries React and Next.js differ.The crucial difference is that React is a JavaScript library, while Next.js is a React framework.So, if you use Next.js, you will inevitably need to use React as well.In terms of image, React has a high degree of freedom because he is a library, and the design etc. are left to the developer.On the other hand, Next.js is a framework, so he has some best practices, and by developing according to the rules, it is possible to proceed with development efficiently. Also, Next.js has a lot of merit, Image optimization,File-based routing,Multiple rendering methods,Fast refresh,Zero config. Thus I should use it.
